## About Flask

**Flask** is a **small and simple web framework for Python** that helps developers build **websites and web applications**.

It is called a **framework** because it provides tools and structure to create web apps easily.

---

## What Flask Is Used For

Flask is used to:

* Build **websites**
* Create **APIs**
* Connect **Python code with web pages**
* Handle **user requests and responses**

When someone visits your website, Flask helps process the request and send the correct page or data back.

---

## Key Features of Flask

**1️⃣ Lightweight**
Flask is small and simple compared to larger frameworks.

**2️⃣ Flexible**
Developers can add only the features they need.

**3️⃣ Easy to Learn**
It is one of the best frameworks for beginners.

**4️⃣ Fast Development**
You can build small projects quickly.

---

## How Flask Works

Basic flow:

1. User opens a website in the browser
2. The request goes to Flask
3. Flask runs Python code
4. Flask returns a web page as a response

Example flow:

```
User Browser → Flask Server → Python Logic → HTML Page → User
```

---

## Components Used in Flask

Common parts in a Flask project:

* **app.py** – main Python file
* **templates/** – HTML files
* **static/** – CSS, images, JavaScript

Example structure:

```
project-folder
│
├── app.py
├── templates
│     └── index.html
└── static
      └── style.css
```
